== Description ==
Bookworm is a robust and flexible WordPress theme, designed by MadrasThemes to help you make the most out of using WooCommerce to power your online store. The design is well suited for Book Store affiliate websites. It is built and comes bundled with most of the advanced features available in most popular eCommerce websites like amazon, flipkart, snapdeal, walmart, alibaba, aliexpress, etc.
